Basic Linux Commands

Command Name Description Example
echo outputs the String the user types after the command echo hello world outputs the string hello world
date This command outputs the current day of the week, day of the month, month, year, and time and time z$ Mon 02 Dec 2019 01:17:56 PM PST
hostname This command outputs the current hostname $ hostname rpi10
arch This command prints machine hardware name $ arch armv7l
cat This command concatenates files and prints on the standard output My Pi Host: rpi10 My Pi User: sp1a Date : zzzzz
rm remove files or directories rm luos0809 This removes the file luos0809 from the directory
scp Copies files between hosts on a network scp luos0809 sp1a@rpi10 This copies the file luos0809 to the location sp1a@rpi10
ls Lists directory contents ls in my home directory displays all the files within it: ls hello tmp
mv Moves/renames files mv hello luos0809 Changes the filename hello to luos0809
passwd Changes passwords for user accounts Type in current password: Type in new password: Re-type new password: (password you type does not reflect on the screen)
